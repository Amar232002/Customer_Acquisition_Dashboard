# Customer Acquisition & Retention Analytics Dashboard

## Overview
Analytics solution to analyze customer trends, engagement patterns, and churn across segments.

## Features
- Customer segmentation using RFM analysis
- Churn prediction and risk analysis
- Revenue optimization insights
- Interactive visualizations

## Tech Stack
- **Python**: Data analysis and processing
- **SQL**: Data querying
- **Power BI**: Dashboard visualization
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn

## Results
- Analyzed 10,000+ customer records
- Identified growth opportunities across 4 segments
- Reduced decision-making time by 30%
- Achieved actionable insights on churn patterns

## Project Structure
```
├── data/
│   ├── raw/              # Original data
│   └── processed/        # Cleaned data
├── notebooks/            # Jupyter notebooks
├── sql/                  # SQL queries
├── screenshots/          # Visualizations
└── dashboard/            # Power BI files
```

## Installation
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

## Usage
1. Run `generate_data.py` to create sample data
2. Open Jupyter: `jupyter notebook`
3. Run notebooks in order: 01_data_cleaning → 02_analysis
4. View results in screenshots folder

## Key Insights
- Champions segment generates 45% of total revenue
- At Risk segment has 23% churn rate
- Email channel has highest customer acquisition
- Top 20% customers contribute 60% revenue
